# medical-camp-management-system
A web-based Medical Camp Registration and Management System to handle patient records, registrations, and camp details efficiently.
Technologies Used
- Python
- Flask
- HTML, CSS

Features
- Patient Registration
- Camp Management
- Data Storage
- Easy Access to Records

 ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
